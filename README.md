# my-agents

AIエージェント向けプロンプト・指示書集

# 運用方針

AIに外注するのではなく、AIを**思考のパートナー**として活用する。

**答えより過程を求める**: 完成品を出させる前に、設計方針・ステップ・選択肢の比較を先に聞く。

**まず自分で書く**: ゼロから生成させず、自分で書いたものをレビュー・添削してもらう。

**ヒントで止める**: 答えを直接教えさせず、デバッグの観点や逆質問で自分の思考を促す。

# ディレクトリ構成

- `AGENTS.md`: 共通指針
- `agents/`: 各役割（ペルソナ）の定義
- `templates/`: 繰り返し使う指示の雛形（調査依頼、添削依頼など）
